CI/CD Pipeline with GitHub Actions

Unified deployment workflow that automatically deploys both frontend and backend on every push to main branch.

name: Deploy Application
on:
push:
branches: [main]
jobs:
deploy-frontend:
runs-on: ubuntu-latest
steps:
- Checkout code
- Deploy to Vercel
deploy-backend:
runs-on: ubuntu-latest
steps:
- Build Docker image
- Deploy to Heroku
